🧱

Как устроена любая программа на Паскале?

Запомни этот шаблон — он одинаковый для всех программ. Меняется только содержимое внутри begin...end.

  • 1
    program Название;
    Имя программы. Любое, без пробелов и русских букв. Чисто как заголовок — на работу программы не влияет.
  • 2
    var имя : тип;
    «Коробочки» для хранения чисел. Сначала имя переменной, потом тип. real — дробные числа (3.5, 7.14). integer — только целые (1, 2, 3). Для треугольника нужен real!
  • 3
    begin ... end.
    Всё между ними — команды программы. Точка в конце обязательна! Без точки Паскаль ругается.
  • 4
    readln(a)
    Программа останавливается и ждёт, пока введёшь число. Введённое число сохраняется в переменную a.
  • 5
    writeln('текст', a)
    Выводит на экран. Текст — в одинарных кавычках. Переменную — без кавычек. Можно через запятую смешивать: сначала текст, потом число.
  • 6
    a := 5 — это присвоение
    := значит «возьми то, что справа, и запомни в переменную слева». Не путай с = — в Паскале это разные вещи!
  • 7
    { это комментарий }
    Всё в фигурных скобках Паскаль игнорирует. Это пометки для себя — можно писать любые объяснения.

Два вида треугольника — два варианта

Ниже представлены оба варианта. Если в задании не уточнено — выбирай Вариант 2, он работает для любого треугольника.

📐 Прямоугольный
2 катета → гипотенуза сама

🔺 Произвольный
вводим 3 стороны

📐 Вариант 1 — Прямоугольный треугольник
➕

Прямоугольный треугольник

У него есть прямой угол (90°) — как угол в комнате. Мы знаем два катета a и b. Гипотенуза c (длинная сторона) считается автоматически.

📏 Площадь = (a × b) ÷ 2

📐 Гипотенуза c = √(a² + b²)  (теорема Пифагора)

🔄 Периметр = a + b + c

💡 Математика в Паскале: квадратный корень — sqrt(...), возведение в квадрат — sqr(...). Например sqrt(sqr(3)+sqr(4)) = √(9+16) = √25 = 5. 
program Triangle_Right;
{ Площадь и периметр прямоугольного треугольника }

var
  a, b : real;  { катеты — два перпендикулярных отрезка }
  c    : real;  { гипотенуза — длинная сторона }
  S    : real;  { S — площадь }
  P    : real;  { P — периметр }

begin
  { Шаг 1: вводим катеты }
  writeln('Введите катет a:');
  readln(a);
  writeln('Введите катет b:');
  readln(b);

  { Шаг 2: гипотенуза по теореме Пифагора }
  c := sqrt(sqr(a) + sqr(b));

  { Шаг 3: площадь — основание умножить на высоту, разделить на 2 }
  S := (a * b) / 2;

  { Шаг 4: периметр — сумма всех трёх сторон }
  P := a + b + c;

  { Шаг 5: выводим ответ. :0:2 = два знака после запятой }
  writeln('Гипотенуза c = ', c:0:2);
  writeln('Площадь S = ', S:0:2);
  writeln('Периметр P = ', P:0:2);

  readln  { чтобы окно не закрылось сразу }
end.
🔺 Вариант 2 — Произвольный треугольник
📏

Произвольный треугольник (формула Герона)

Когда треугольник не обязательно прямоугольный — вводим три стороны. Площадь считаем по формуле Герона. Греческий математик придумал её 2000 лет назад!

➡️ Периметр P = a + b + c

➡️ Полупериметр p = P ÷ 2  (нужен для формулы ниже)

🌿 Площадь S = √(p × (p−a) × (p−b) × (p−c))

💡 Зачем полупериметр? Формула Герона работает только через полупериметр — так её придумали. Сначала считаем p = (a+b+c)÷2, потом подставляем в формулу. Всё! 
program Triangle_Any;
{ Площадь и периметр любого треугольника — формула Герона }

var
  a, b, c : real;  { три стороны }
  p       : real;  { полупериметр (половина периметра) }
  S       : real;  { площадь }
  P       : real;  { периметр }

begin
  { Шаг 1: вводим три стороны }
  writeln('Введите сторону a:');
  readln(a);
  writeln('Введите сторону b:');
  readln(b);
  writeln('Введите сторону c:');
  readln(c);

  { Шаг 2: периметр и полупериметр }
  P := a + b + c;
  p := P / 2;

  { Шаг 3: формула Герона }
  S := sqrt(p * (p - a) * (p - b) * (p - c));

  { Шаг 4: выводим результат }
  writeln('Площадь S = ', S:0:2);
  writeln('Периметр P = ', P:0:2);

  readln
end.

Разбор ошибок из кода на уроке

На скриншоте с урока была незаконченная программа. Вот все ошибки — чтобы не повторить:

❌
wirte('S=', S)  →  writeln('S=', S)

Опечатка в команде. Паскаль не угадывает — нужно писать точно. Также лучше использовать writeln (с ln в конце), тогда каждый результат будет на новой строке.

❌
S := a*b  →  S := (a*b)/2

Формула площади прямоугольника! Треугольник — это «половина прямоугольника», поэтому нужно разделить на 2.

❌
P := (a*b)*2  →  P := a + b + c

Это периметр прямоугольника. Периметр любой фигуры = сумма всех сторон. У треугольника три стороны — складываем все три.

❌
var a, b, s: integer  →  var a, b, s: real

Тип integer — только целые числа. Стороны могут быть дробными (3.5, 2.7...). Нужен тип real, иначе дробная часть просто обрежется и ответ будет неправильным.

❌
Переменная c (гипотенуза) не объявлена в блоке var

Все переменные, которые используются в программе, обязательно нужно сначала объявить в var. Иначе Паскаль скажет «что это такое?» и не запустит программу.
